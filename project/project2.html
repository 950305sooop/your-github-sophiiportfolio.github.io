VoiceNote

The app allows user organize and shares their To-Do list by one tap on home widget, it convert voice to text and updates to both users' side.

Role: UX Research, Wireframes, UI, Prototype, AI prompt, code
Timeline: August 14 — Semtemper 2, 2025
Type: Self-initiated project

Project Overview

UX/UI Design • Effiency • AI Assistant

  • Tools: Figma, HTML, Supabase, ChatGPT, Claude, Netlify, Notion.
  • Netlity Link(in progress work): VoiceNote Link

Voice Note is a concept app I designed to help people capture and share to-do lists in the most natural way, which is by speaking. The app automatically converts voice into text, organizes tasks, and allows sharing with close friends via widget.

The Problem

• To-do tasks gets override easy in the messages.
• Quick thoughts/tasks get lost if not written down immediately.
• Existing note apps are too slow
(open → type → save → share).

The Goal

• Capture tasks instantly, effortlessly.
• Make the To-do list stand out on the device.
• Share selectively with trusted friends.

Target Users

Who: Busy students, workers, or people with ADHD who need fast capture.

Needs:
• Capture tasks instantly, effortlessly.
• Automatic reminders through shared widget
• Easy to organize the tasks.

Initial Ideas

How might we capture the quick thoughts? How might we make sure important small tasts get done?

I was texting my boyfriend and reminding him what we need to buy from Costco. I quickly realized that shopping list gets override by other texts, making it hard to track. We both likes to use voice message because it's faster and it frees your hand from typing. But at the same time, it takes more time to listen than reading the texts. The idea of VoiceNote formed, converting voices to texts and shares it automatically.
